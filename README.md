# Weather App

This is a simple weather application built using React and Vite that fetches weather data from the OpenWeatherMap API and displays it to the user.

## Features

- Users can search for weather information by entering a city name.
- Displays current temperature, humidity, wind speed, and weather icon.
- Error handling for API requests.

## Technologies Used

- React
- Vite
- OpenWeatherMap API
- JavaScript (ES6+)
- HTML/CSS

## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/lloydsebastian/Weather-App.git

2. Navigate to the project directory:
  cd weather-app

3. Install dependencies:
  npm install

4. Create a .env file in the root directory and add your OpenWeatherMap API key:
  VITE_APP_ID = your_api_key_here

5. Start the development server:
  npm run dev

6. Open your browser and visit the URL provided in the terminal (usually http://localhost:5173) to view the app.)

## Usage

1. Enter a city name in the search bar and press Enter or click the search icon.
2. Weather information for the specified city will be displayed.
